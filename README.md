# Course Management System Java project
# Description
A Java-based course management system designed to help university students plan and register for the right courses each semester. The system allows students to view available courses, check prerequisites, track completed subjects, and identify which courses they are eligible to take next.

#Features
- View all available courses
- Check course prerequisites  
- Track completed courses
- Get recommendations for next semester
- Student registration system

#Technologies Used
- Java Programming language
- Eclipse IDE

#How to run
1. Download or clone the respitory
2. Open in Java IDE
3. Run the main class
